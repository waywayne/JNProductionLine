## 🔍 SN 码格式问题

### 读取到的 SN 码

```
6371601151000IW00MN
```

### 格式分析

| 位置 | 预期 | 实际 | 状态 |
|------|------|------|------|
| **0-2** | `637` (产品线) | `637` | ✅ 正确 |
| **3** | `1` (工厂) | `1` | ✅ 正确 |
| **4-8** | `60115` (日期 2026-01-15) | `60115` | ✅ 正确 |
| **9** | `1` (产线) | `1` | ✅ 正确 |
| **10-14** | `00010` (流水号，纯数字) | `000IW` | ❌ **错误** |
| **15-18** | `XXXX` (校验码) | `00MN` | ❌ **可能错误** |

### 问题点

1. **流水号包含字母**: `000IW` 应该是 `00010` 等纯数字
2. **可能的原因**:
   - 字符显示错误（`I` 可能是 `1`，`W` 可能是其他字符）
   - FLASH 数据损坏
   - 写入时使用了错误的格式
   - 测试数据或占位符

## ✅ 解决方案

### 1. 添加 SN 码格式验证

在 `_autoTestReadSN()` 中添加格式验证逻辑：

```dart
// 验证SN码格式
if (!_snManager.validateSN(snCode)) {
  _logState?.warning('⚠️ SN码格式无效或校验失败');
  _logState?.info('   读取的SN: $snCode');
  _logState?.info('   SN码长度: ${snCode.length} (应为19位)');
  
  // 检查是否包含非法字符
  final hasInvalidChars = !RegExp(r'^[0-9A-Z]+$').hasMatch(snCode);
  if (hasInvalidChars) {
    _logState?.warning('   ⚠️ SN码包含非法字符（应只包含数字和大写字母）');
  }
  
  _logState?.info('   将在下一步写入新的有效SN码');
  
  // 清空当前设备标识，强制生成新SN
  _currentDeviceIdentity = null;
  return true;
}
```

### 2. SN 码验证规则

**文件**: `lib/services/sn_manager_service.dart`

```dart
bool validateSN(String sn) {
  // SN 格式: PPP-F-YMMDD-L-BBBBB-SSSS
  // 实际存储: PPPFYMMDDLBBBBBSSSS (19位)
  if (sn.length != 19) return false;
  
  // 提取基础 SN 和校验码
  final baseSN = sn.substring(0, 15);
  final checksum = sn.substring(15);
  
  // 验证校验码
  final calculatedChecksum = _calculateChecksum(baseSN);
  return checksum == calculatedChecksum;
}
```

## 🎯 验证规则

### 1. **长度检查**

```dart
if (sn.length != 19) return false;
```

SN 码必须是 19 位。

### 2. **字符检查**

```dart
final hasInvalidChars = !RegExp(r'^[0-9A-Z]+$').hasMatch(snCode);
```

SN 码只能包含：
- ✅ 数字 `0-9`
- ✅ 大写字母 `A-Z`
- ❌ 小写字母
- ❌ 特殊字符
- ❌ 空格

### 3. **校验码验证**

```dart
final baseSN = sn.substring(0, 15);
final checksum = sn.substring(15);
final calculatedChecksum = _calculateChecksum(baseSN);
return checksum == calculatedChecksum;
```

校验码必须与计算值匹配。

## 🔧 常见无效 SN 示例

| SN 码 | 问题 | 说明 |
|-------|------|------|
| `6371601151000IW00MN` | 包含字母 `I`, `W` | 流水号应为纯数字 |
| `637160115100001` | 长度不足 | 应为 19 位 |
| `6371601151000010000abc` | 长度超出 | 应为 19 位 |
| `637-1-60115-1-00001-0000` | 包含分隔符 | 存储格式不应有分隔符 |
| `6371601151000010001` | 校验码错误 | 校验码不匹配 |

## ⚠️ 注意事项

### 1. **不要强制使用无效 SN**

无效的 SN 码会导致：
- ❌ 数据库查询失败
- ❌ 追溯困难
- ❌ 质量管理问题

### 2. **记录所有无效 SN**

建议将无效 SN 记录到单独的日志文件，便于分析：
```dart
_logState?.error('发现无效SN: $snCode', type: LogType.device);
```

### 3. **通知用户**

如果频繁出现无效 SN，应该：
- 提示用户检查设备
- 可能需要固件更新
- 检查 FLASH 健康状态
